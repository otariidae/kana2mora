# kana2mora

Divide a string of Japanese Kana into mora

## Installation

```
$ npm i kana2mora
```

## Usage

```ts
const { katakana2mora } = require("kana2mora")

katakana2mora("チョコレート") // ["チョ", "コ", "レ", "ー", "ト"]
```
